# 📊 Annual Sales Trend and Growth Analysis Dashboard

## 🚀 Project Overview
The **Annual Sales Trend and Growth Analysis Dashboard** is a data analytics project designed to analyze sales performance, profit trends, and customer contributions across different regions and time periods.

This project demonstrates a **complete data analysis workflow** including data extraction, cleaning, analysis, and visualization using modern data analytics tools.

The final output is an **interactive Power BI dashboard** that helps businesses understand their sales performance and make **data-driven decisions**.

---

## 🛠 Technologies Used

| Tool | Purpose |
|-----|------|
| **SQL** | Data querying and extraction |
| **Python (Pandas)** | Data cleaning and preprocessing |
| **Microsoft Excel** | Data preparation and verification |
| **Power BI** | Data visualization and dashboard creation |
| **Jupyter Notebook** | Data analysis environment |

---

## 🔄 Project Workflow

1️⃣ **Data Collection**  
Sales dataset collected for analysis.

2️⃣ **Data Cleaning (Python & Pandas)**  
Data preprocessing performed in **Jupyter Notebook** including:
- Removing duplicates
- Handling missing values
- Data formatting
- Preparing structured dataset

3️⃣ **Data Querying (SQL)**  
SQL queries used to analyze:
- Total sales
- Customer contribution
- Regional performance
- Profit trends

4️⃣ **Data Preparation (Excel)**  
Excel used for quick data inspection and preparation.

5️⃣ **Data Visualization (Power BI)**  
An interactive dashboard created to visualize sales insights.

## 📊 Dashboard Highlights

The dashboard provides multiple visualizations and analytical features to help understand sales performance effectively.

### 📌 Key Performance Indicators (KPIs)
- **Total Sales** – Displays overall revenue generated.
- **Total Profit** – Shows total profit earned from sales.
- **Total Orders** – Indicates the number of completed orders.

These KPIs provide a **quick overview of overall business performance**.

### 📈 Sales and Profit Trend Analysis
- **Monthly Sales Trend Line Chart** to track sales growth throughout the year.
- **Profit Trend Visualization** to identify months with higher or lower profitability.
- Helps detect **seasonal patterns and growth trends**.

### 👥 Customer Performance Analysis
- Bar chart showing **sales contribution by customers**.
- Helps identify **top-performing customers**.
- Supports businesses in focusing on **high-value customers and retention strategies**.

### 🌍 Regional Sales and Profit Comparison
- Visualization comparing **profit generated from different regions**.
- Helps identify **strong and weak performing regions**.
- Useful for **regional business strategy planning**.

### 🔍 Sales vs Profit Relationship
- **Scatter plot visualization** showing the relationship between sales and profit.
- Helps analyze **profitability patterns and outliers**.
- Useful for understanding which sales generate the **highest profit margins**.

### 🎛 Interactive Dashboard Filters
Users can interact with the dashboard using filters such as:

- **Year Filter** – Compare sales performance across years.
- **Region Filter** – Analyze performance for specific regions.
- **Customer Filter** – Focus on individual customer contributions.

These filters make the dashboard **dynamic and easy to explore**.

### 📊 Business Insights
The dashboard enables users to:
- Track **sales growth over time**
- Identify **top customers and profitable regions**
- Understand **sales and profit relationships**
- Support **data-driven decision making**
